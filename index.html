Analizė

Puslapio autorystė

Kriterijus Įvertinimas
Nurodytas puslapio autorius (pvz., metaduomenyse, puslapiuose)
Nurodyta sukūrimo/ atnaujinimo data (pvz., metaduomenyse, puslapiuose)
Nurodyta kontaktinė informacija
Nurodyta puslapio institucija

Struktūra

Kriterijus Įvertinimas
Tinklalapis turi tinklalapio antraštę (Head) (AA-daugumoje puslapių / A-keliuose puslapiuose / viename puslapyje / nenurodytas)
Tinklalapis turi pavadinimą (Title) (AA-daugumoje puslapių / A-keliuose puslapiuose / viename puslapyje / nenurodytas)
Vieningas puslapio stilius (AA-dauguma puslapių panašūs /A- keletas puslapių panašūs / A-dauguma puslapių skiriasi)
Yra turinys (nuorodos į daugumą svetainės puslapių / nuorodos į keletą svetainės puslapių / turinys be nuorodų / turinio nėra)
Naudojami raktažodžiai (daugumoje puslapių skirtingi / keliuose puslapiuose skirtingi / visuose puslapiuose vienodi / nenaudojami)
Tinklalapio navigacija aiški

Turinio prieinamumas (Accessibility)

Kriterijus Įvertinimas
Puslapis atrodo taip pat naudojant skirtingas naršykles
Puslapis pritaikytas skirtingoms monitoriaus rezoliucijoms
Puslapis turi galimybę pakeisti turinio kalbą
Puslapis pritaikytas neįgaliesiems
Naudojama kontekstinė pagalba
Tinklalapis veikia išjungus javascript
Tinklalapis neišsidarko atjungus CSS

Turinio autorystė

Kriterijus Įvertinimas
Naudojamos medžiagos autorystė (didžioji dauguma priklauso puslapio kūrėjams / puslapių kūrėjų ir svetimų autorių medžiagos santykis yra apylygis / didžioji dauguma priklauso svetimiems autoriams)
Pateikiama medžiaga nėra nelegali
Turinys atitinka pavadinimą ir paskirtį

Lengvas skaitomumas

Kriterijus Įvertinimas
Tinkamas šrifto tipas, dydis, spalva
Galimybė proporcingai keisti šrifto dydį
Išskirtos nuorodos
Efektyvus didesnės apimties turinio suskaidymas

Audio / video / grafika (multimedia)

Kriterijus Įvertinimas
Multimedija taikoma turiniui atskleisti
Multimedija nesulėtina puslapio veikimo
Naudojama vaizdo ar garso kontrolė – sustabdymas, paleidimas (jei puslapyje naudojami tokio tipo objektai)
Paveikslai tinkamai aprašyti HTML lygmenyje
Puslapiai pritaikyti spausdinimui
Paveikslai tinkamo formato ir kokybės

Veikimas

Kriterijus Įvertinimas
Įvedus duomenis grąžinamas atsakymas (jei naudojamos formos)
Formos grąžina teisingus duomenis
Nėra neatsidarančių nuorodų
Nėra klaidingai nukreipiančių nuorodų
Puslapyje nerodomos sisteminės klaidos (serverio, turinio valdymo sistemos ir kt.)

Taisyklinga kalba

Kriterijus Įvertinimas
Nėra sintaksinių ar gramatinių klaidų
Nevartojamas žargonas, keiksmodžiai
Naudojamas lietuviškas šriftas (jei svetainė lietuviška)

Atnaujinimas

Kriterijus Įvertinimas
Nurodyta atnaujinimo data
Nurodyti atnaujinti objektai
Atnaujinimai vykdomi pakankamai dažnai
Galimybė prenumeruoti naujienas

Našumas

Kriterijus Įvertinimas
HTML dydis (atitinkamai pagal tinklalapį, pageidautina <50kb)
Bendras dydis (atitinkamai pagal tinklalapį, pageidautina < 150kb, priimtina <500kb)
Užklausų kiekis (viso tinklalapio parsiuntimui, pageidautina iki 50)
Generavimo laikas (pageidautina iki 2sek., priimtina iki 4-5 sek.)
CSS ir Javascript failų kiekis (pageidautina iki 3, priimtina iki 10)
Galimybė siųsti naršyklei suarchyvuotą HTML (naudojamas Gzip ar ne)
Ten kur prasminga, naudojamas AJAX (pvz., vertinimui, balsavimui, komentavimui ir pan.)

Standartų laikymasis

Kriterijus Įvertinimas
Validus HTML (validator.w3.org)
Validus CSS (jigsaw.w3.org/css-validator/)
Javascript be klaidų (reikia žiūrėti į naršyklės javascript konsolę)
Lentelių naudojimas formatavimui (nenaudojamos, naudojamos saikingai/nesaikingai)

Tinklapių marketingo priemonės. Tinklapių optimizavimas paieškos sistemoms SEO (Search Engines Optimization)

Kriterijus Įvertinimas
Tinklalapio rangas google sistemoje
Vidinės puslapių nuorodos yra prasminės (pvz. www.zalgiris.lt/apie-komanda.html)
Naudojami raktažodžiai ir aprašymas metaduomenyse
Raktažodžiai ir aprašymas metaduomenyse naudojami saikingai (t.y., nespaminama raktažodžiais, iki 10 raktažodžių)
Raktažodžiai susiję su turinių ir dalį jų galima rasti tekstuose
Nuorodos ir paveiksliukai turi prasminius alternatyvius aprašymus (naudojant alt arba title atributus)
Laikomasi semantinio žiniatinklio taisyklių (pvz., lentelės naudojamos lenteliniams duomenims, sąrašinės struktūros – sąrašams, antraštės – tik tikroms antraštėms ir t.t.)
Naudojami žymių (raktažodžių) debesys (tag clouds) (jei naudojami – ar naudojami prasmingai)
Tinklalapių turinys (tekstai, flash objektai) pasiekiamas ir išjungus javascript’ą (kartais neteisingai panaudojamas AJAX, flash objektai pasiekiami tik per javascript’ą)
Egzistuoja sudarytas svetainės žemėlapis paieškos sistemoms (paprastai - sitemap.xml)
Naudojamos ar ne “juodosios technologijos” (pvz., raktažodžiai yra puslapyje, bet pažymėti fono spalva, kad juos matytų tik paieškos sistemos, o žmonės – ne ir pan.)
